<h2 id="uso">Como Usar</h2>
<h2>
<ol>
  <li>Digite o que quiser traduzir.</li>
  <li>A linguagem de tradução é fixa, então você sempre vai traduzir de pt-br para a linguagem que selecionar.</li>
  <li>Selecione o idioma para qual deseja traduzir.</li>
  <li>Clique no botão: <img src="https://github.com/Jeff-Benini/Tradutor/blob/main/img/Bot%C3%A3o%20tradutor.jpg?raw=true"/> para traduzir.</li>
  <li>Caso queira traduzir por voz clique no botão: <img src="https://github.com/Jeff-Benini/Tradutor/blob/main/img/Bot%C3%A3o%20microfone.jpg?raw=true"/> e o tradutor irá traduzir o que você falar.</li>
</ol>
